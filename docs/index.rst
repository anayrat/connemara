Welcome to Connemara's documentation!
=====================================

The connemara replication system is meant to federate different databases  under one common database, in different schemas.
The original purpose was to federate several databases in one place for BI tools
to join data from several data sources without the performance and complexity of
Foreign Data Wrappers.
